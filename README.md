## Dataset ðŸ“Š
The Iris dataset contains 150 samples with 4 features:
| Sepal Length (cm) | Sepal Width (cm) | Petal Length (cm) | Petal Width (cm) | Species         |
|--------------------|-------------------|--------------------|-------------------|------------------|
| 5.1                | 3.5               | 1.4                | 0.2               | Iris-setosa      |
| 6.7                | 3.0               | 5.2                | 2.3               | Iris-virginica   |

Three target classes:
- Iris-setosa
- Iris-versicolor 
- Iris-virginica
